Welcome to Lootster

Lootster is a free play-to-earn lootbox lottery game on the TON blockchain, where players can earn TON and LTR tokens. Starting as a fair game, it will grow into a transparent gaming ecosystem as outlined in our roadmap.

Features

Fair Play via smart contracts

Transparency on the TON blockchain

Real rewards with market value tokens

Ease of Use with TON wallet integration

Games

Tokenomics

Ton

An exclusive reward token, obtainable only by opening chests.

LTR

A utility token used for rewards, gameplay, and liquidity.

50% Cheast reward

25% Core contributors

10% Providing liquidity

10% Airdrop for NFT holders

5% Early supportest

Roadmap

Stage 1

Closed beta (Q4 2024)

  • Get rewards in TON and LTR tokens from chests in Lootster game
  • Leaderboards for LTR and TON tokens: Daily, weekly and monthly
  • Referral system
  • Daily and Basic quests
  • NFT Achievements for group quests

Stage 2

Release (Q1 2025)

  • World Map
  • Free NFT closed beta airdrop
  • "Bomb It Up" PvP game release
  • New UI system with world map
  • Token Generation Event (TGE)
  • Token swap

Stage 3

Update (Q2 2025)

  • Crack the Egg game release
  • In-game token swap
  • NFT boosters for Lootster game
  • More Languages support
  • CEX listing
  • Open game data
  • Airdrop for various NFT holders
